常量/变量
**常量**：一旦赋值，就不可再改变，就是不能对它重新赋值。**Python不存在常量**
**字面常量**：一个单独出现的量，未赋值给任何变量或常量。例如print('hello world')
对python没有太大意义，只是需要了解，其他语言会有常量

**变量**：是一个名字，在赋值符号的左边，这个名字可以指代赋值符号右边的内容
	例如：i = 3
			print(i)

### 类型系统 ###
	* Python是一个强类型语言
	* Python是动态类型语言
		**强类型**：指的是不同类型之间不能相互计算：运算的时候会做类型检查
		例如：4+4 OK
		4+'4' No
		字符类型必须一样

		**动态类型语言**：变量可以重新赋值为其他类型
				注：给变量赋值什么类型的值，它就是什么类型。
			整形
			例如：i = 4
		      type(i) #查看i的类型。
			Python中的整数原则上是没有长度限制的，实际上是受物理内存大小的限制。
			
			浮点型
			例如：i = 1.23
					type(i)
					float
			Python中float是有长度限制的，会有益处，而且Python中float是会损失精度的（精度丢失）。
				财务、金融以及科学计算，不能使用Python的float，可以改用int。
				
			布尔类型
			例如：i = True
					布尔类型只有True和False。
					
			None类型（一个空引用）
			例如：i = None
				只有单独的None，只有一个值。
			
			字符串（str是组合类型，不是基础类型）
				由基础类型构建而成的。
			
			Python当中没有单独的byte类型和X类型，基础类型为int和float,两个特殊的基础类型bool和None，不过也没有严格的区分
			Python当中一切都是引用，所以一切都是指针
### 运算符 ###
	## 算术运算符 ##
		算术运算符只能对int和float运算
		例如：1 + 1
		      1 + 1.2   (由于是强制类型的，所以会有一个隐式转换，把 1 装换成了float类型，大部分语言都会有隐式转换)
			+、-、*、/
			/：Python2 是取整除（3 / 2 = 1，绝大部分语言都是取整除），Python3 是自然除（3 / 2 = 1.5）
					Python中如果把任意一个整数转换成为float类型，就可以达到自然除的目的
					除法的时候0不能作为除数
	## 比较运算符 ##
		<、>、==、!=、>=、<=
